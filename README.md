# Food Ordering System

## Overview
A web-based Food Ordering System developed using Node.js, Express.js, MySQL, HTML, CSS, JavaScript, and Bootstrap.

The system provides separate customer and admin modules with secure JWT authentication.

## Features

### Customer
- User Registration and Login
- Browse Food Menu
- Search Food Items
- Add Items to Cart
- Place Orders
- View Order History

### Admin
- Add Food Items
- Upload Food Images
- Delete Food Items
- View All Orders
- Update Order Status

## Technologies Used
- Node.js
- Express.js
- MySQL
- JWT Authentication
- Bootstrap 5
- HTML/CSS/JavaScript

## Security Features
- JWT Authentication
- Password Hashing (bcrypt)
- Role-Based Access Control
- Protected Routes
